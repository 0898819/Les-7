HTML/CSS EINDOPDRACHT

REFLECTIE

Tijdens dit keuzevak heb ik een opfriscursus gehad betreft HTML/CSS. In de eerste twee jaar heb ik een aantal workshops gevolgd waarin HTML/CSS werd aangeleerd. Ook heb ik tijdens mijn 3e-jaars stage veel met HTML/CSS gewerkt. Veel van deze stof was weggezakt, het was dus fijn om dit weer gestructureerd naar voren te halen. Ik vond het fijn om alles weer vanaf het begin uitgelegd te krijgen. Verder vond ik het heel fijn om met GitHub te werken. Doordat dit goed werd uitgelegd in les 2 had ik bij mijn minor al een voorsprong op de rest.

ACTIENPUNTEN

Om in de toekomst meer te kunnen schrijven met HTML/CSS, en uitdagendere code te schrijven wil ik een aantal dingen gaan doen. Ik wil vooral veel blijven doen; proberen dingen na te maken, effecten na te bootsen en bestaande elementen uit websites halen en toepassen in mijn eigen werk. Verder wil ik mezelf gaan verbreden door ook te kijken naar JAVA en PHP om wat uitdagendere ontwerpen uit te kunnen werken omdat wanneer je met enken HTML/CSS werkt dit soms wat beperkt kan zijn. Ook wil ik me gaan focussen op het schrijven van code die nog cleaner en overzichtelijker is.

TO BUILD
